📄 CSV File Upload

                    

                        
                        💡 CSV Format (3 columns):

                        Column 1: Dimension name (e.g., "Region")

                        Column 2: Filter values, comma-separated (e.g., "East, West" creates a metric with BOTH values)

                        Column 3: Follower emails, comma-separated (e.g., "user@example.com, admin@example.com")


                        Example CSV:

                        
                        Region,East,user1@example.com

                        Region,"East, West",user2@example.com

                        Department,Sales,user3@example.com,user4@example.com

🔍 Zero Follower Metrics

                
Find metrics within a definition that have no followers

                
            

            
            

                
                

                    
🔗 Site Connection

                    

                        
                    

                    

                        
                    

                    

                        
                    

                    

                        

                            
                        

                        

                            
                        

                    

                    

                        

                            
                        

                        

                            
                        

                    

                


                
                

                    
📊 Definition Selection

                    

                        
                        Enter the ID of the Pulse definition to analyze for metrics with zero followers
                    

                


                
                

                    
⚙️ Analysis Options

                    

                        
                        If unchecked, analysis will be faster by only checking if metrics have followers (not counting them). Follower counts are only used for display/reference.
                    

                


                
                

                    
🗑️ Deletion Options

                    

                        
                        ⚠️ WARNING: This will permanently delete all non-default metrics that have zero followers. Default metrics cannot be deleted and will be skipped.
                    

                


                

                    
                    
                

            

        


        
        

            

                
🧹 Remove All Followers

                
Remove all followers from all non-default (scoped) metrics for a given definition. The default metric is left unchanged.

                
            

            
            

                
                

                    
🔗 Site Connection

                    

                        
                    

                    

                        
                    

                    

                        
                    

                    

                        

                            
                        

                        

                            
                        

                    

                    

                        

                            
                        

                        

                            
                        

                    

                


                
                

                    
📊 Definition Selection

                    

                        
                        Enter the ID of the Pulse definition. All followers will be removed from every non-default (scoped) metric under this definition. The default metric is not modified.
